## Run Local

Clonar el projecto

```bash
  git clone https://github.com/DeveloFast/ecofuel.git
```
Acceda a la carpeta
```bash
  cd ecofuel
```



Instale dependencias 

```bash
  npm install
```


Para correr el proyecto ejecute el siguiente comando 

```bash
  npm run dev
```
